Windows 10 21H2 inches closer to launch with these new features

Windows 10 21H2 is now a step closer to release as Microsoft has finally started testing its features with testers in the Release Preview Channel of the Windows Insider program. Alongside Windows 11, Windows 10 version 21H2 is now set to launch in October-November as the second feature update of 2021.

Windows 11 Build 22000.160 (KB5005189) released with new feature

Description, 20.8.2021

Windows 11 Build 22000.160 (KB5005189) released with new feature

Windows 11 Build 22000.160 (KB5005189) has been released for Windows Insiders (Dev and Beta) with several bug fixes and improvements. Alongside the new preview build, Microsoft has also started pushing an upgraded Clock app which brings a new feature called “Focus Sessions”.

Microsoft shows off MS Paint with new design and dark mode on Windows 11

Description, 19.8.2021

Microsoft shows off MS Paint with new design and dark mode on Windows 11

Alongside Snipping Tool, Windows Calculator, and Mail & Calendar app, everyone’s favourite and popular MS Paint is also seeing a redesign in Windows 11. The classic Paint tool is being refreshed with WinUI, Fluent Design, and modern controls to make it look similar to the new apps for Windows 11.
